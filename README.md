# labelPrint

수축율 시험 라벨 출력을 자동화하기 위한 Excel VBA 매크로 파일입니다.

접수번호 목록을 기준으로 시험번호 텍스트 파일과 `start.xlsx` 정보를 읽어 라벨 양식에 값을 채우고, 건별 확인 출력 또는 전체 자동 출력을 수행합니다.

## 파일 구성

| 경로 | 설명 |
| --- | --- |
| `labelPrint2_v0.1_20260415.xlsm` | 최신 작업용 Excel 매크로 파일 |
| `labelPrint2_v0.1_20260406.xlsm` ~ `labelPrint2_v0.1_20260414.xlsm` | `labelPrint2` 이전 백업 버전 |
| `labelPrint_v1.1_20260118.xlsm`, `labelPrint_v1.2_20260202.xlsm` | 기존 `labelPrint` 버전 |
| `src/labelPrint2_v0.1_20260415.bas` | 최신 VBA 모듈 백업 소스 |
| `src/labelPrint_v1.2_20260202.bas` | 기존 v1.2 VBA 모듈 백업 소스 |

## 주요 매크로

최신 버전 기준 주요 실행 매크로는 다음과 같습니다.

| 매크로 | 설명 |
| --- | --- |
| `출력_개별확인3` | 각 라벨 출력 전 확인창을 띄우는 방식 |
| `출력_전체자동3` | 확인 없이 전체 라벨을 자동 출력하는 방식 |
| `자동_양식_출력_20260415` | 라벨 데이터 생성과 출력의 핵심 처리 루틴 |

기존 v1.2 파일에는 `출력_개별확인`, `출력_전체자동`, `자동_양식_출력_20260117` 매크로가 포함되어 있습니다.

## 사용 준비

매크로 파일과 같은 폴더에 다음 파일/폴더가 필요합니다.

| 항목 | 설명 |
| --- | --- |
| `start.xlsx` | 접수번호별 업체명, 품명 등 기본 정보를 조회하는 파일 |
| `testNumber/2025/{접수번호}.txt` | 접수번호별 시험 항목과 수축율 조건을 담은 텍스트 파일 |

Excel에서 매크로 사용을 허용한 뒤, 최신 파일 `labelPrint2_v0.1_20260415.xlsm`을 열어 실행합니다.

## 동작 개요

1. `접수번호` 시트의 B열에서 접수번호를 순서대로 읽습니다.
2. 접수번호를 `H231-25-22028` 형태로 변환합니다.
3. `testNumber/2025/{접수번호}.txt`에서 수축율 조건, 시험 코드, 시료 수를 읽습니다.
4. `start.xlsx`에서 업체명과 관련 정보를 조회합니다.
5. `양식` 시트에 라벨 정보를 채웁니다.
6. 선택한 출력 방식에 따라 개별 확인 또는 전체 자동 출력합니다.

## 버전 관리 방식

- `.xlsm` 파일은 실제 실행용 파일입니다.
- `src/*.bas` 파일은 VBA 모듈을 별도로 백업한 소스입니다.
- Excel이 자동 생성하는 `~$*.xlsm` 임시 잠금 파일은 Git에서 제외합니다.
