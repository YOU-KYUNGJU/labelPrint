# Changelog

이 문서는 현재 저장소에 포함된 과거 `.xlsm` 파일과 `src/*.bas`로 export된 VBA 소스를 기준으로 정리한 변경 이력입니다.

## 분석 기준

- `.xlsm` 파일은 바이너리 파일이므로 버전별 상세 코드 diff는 직접 확인하기 어렵습니다.
- 각 `.xlsm` 파일은 내부 `xl/vbaProject.bin` 크기와 SHA-256 해시 앞 12자리를 비교했습니다.
- 시트 구성은 모든 버전에서 `접수번호`, `양식`, `Sheet3`로 동일했습니다.
- 상세 코드 변경점은 export된 두 소스 파일을 비교했습니다.
  - `src/labelPrint_v1.2_20260202.bas`
  - `src/labelPrint2_v0.1_20260415.bas`

## 버전별 변경 이력

### labelPrint2 v0.1 20260415

최신 실행 파일입니다. `src/labelPrint2_v0.1_20260415.bas` 소스가 함께 보관되어 있어 코드 수준의 변경 내용을 확인할 수 있습니다.

#### 변경됨

- 메인 실행 루틴이 `자동_양식_출력_20260415`로 갱신되었습니다.
- 실행용 래퍼 매크로가 다음 이름으로 분리되었습니다.
  - `출력_개별확인3`
  - `출력_전체자동3`
- `start.xlsx` 조회 결과에서 납품업체 값이 `CPLB`인 경우 접수번호 앞에 `!`를 붙이도록 처리했습니다.
- 시험조건 표시값을 그대로 쓰지 않고, 첫 번째 쉼표 다음부터 콜론 전까지의 문자열만 사용하도록 정리했습니다.
  - 예: `AAA, BBB : CCC` → `시험조건 BBB`
  - 콜론이 없으면 첫 번째 쉼표 다음부터 끝까지 사용합니다.
- 시험조건 원문에 `뒤틀림`이 포함된 항목은 라벨 출력 대상에서 제외합니다.
- 가공된 시험조건이 직전 출력 조건과 같으면 중복 출력하지 않도록 건너뜁니다.
- 출력 대상 조건을 비교하기 위해 `prevPrintedB1`, `rawB1`, `printB1` 처리 변수가 추가되었습니다.

#### 유지됨

- `접수번호` 시트 B열을 순회하며 접수번호를 읽습니다.
- `testNumber/2025/{접수번호}.txt`에서 수축율 시험 조건, 시험 코드, 시료 수를 읽습니다.
- `start.xlsx`에서 업체명, 품명 등 라벨에 들어갈 기본 정보를 조회합니다.
- `접수번호` 시트 C열의 수동 시료번호 지정 기능을 유지합니다.
- 개별 확인 출력과 전체 자동 출력 방식을 모두 지원합니다.
